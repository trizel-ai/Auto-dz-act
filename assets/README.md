## Design Specifications

### Colors
- **Primary**: #3FA9F5 (Blue accent)
- **Background**: #0B1220 → #070B14 (Dark gradient)
- **Text**: #FFFFFF (White, varying opacity)

### Typography
- **Font stack**: `ui-sans-serif, system-ui, -apple-system, Segoe UI, Roboto, Arial, sans-serif`
- **No external fonts** — uses system fonts for reliability

### Accessibility
- Proper ARIA labels included
- High contrast ratios
- Scalable vector format
- Works on light/dark backgrounds
